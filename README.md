## Que es este proyecto?

**Blog-AI** es una guia informativa interactiva, completamente en espanol, pensada para ayudarte a pasar de usuario basico de IA a construir agentes autonomos que trabajan mientras dormis.

El proyecto cubre todo lo que necesitas saber sobre **Claude** (de Anthropic): conceptos, comandos, prompts listos para copiar, herramientas recomendadas y como automatizar tareas reales.

---

## Contenido del blog

El blog esta organizado en **6 modulos tematicos**:

| # | Modulo | Descripcion |
|---|--------|-------------|
| 1 | **Fundamentos** | Que es Claude, modos de uso (Chat, Cowork, Claude Code) y conceptos clave |
| 2 | **Diseno Web** | Como usar Claude para crear interfaces, webs y proyectos visuales |
| 3 | **Marketing y Contenido** | Prompts para copywriting, redes sociales, emails y estrategias |
| 4 | **Agentes y Automatizacion** | WhatsApp AgentKit, control remoto de PC, agentes en la nube 24/7 |
| 5 | **Productividad** | Obsidian como memoria, Skill Seekers, Cowork y skills propios |
| 6 | **Referencia** | Tabla completa de comandos, instalaciones y prompts listos para copiar |

---

## Funcionalidades interactivas

- Busqueda en tiempo real con resaltado de resultados
- Sistema de favoritos persistido con localStorage
- Filtros por tipo: comandos, prompts, conceptos, favoritos
- Copiar con un clic todos los prompts y comandos
- Acordeones colapsables para una navegacion limpia

---

## Tecnologias utilizadas

- **HTML5 + CSS3** sin frameworks externos
- **JavaScript vanilla** sin dependencias
- **localStorage** para persistir preferencias y favoritos
- **GitHub Pages** despliegue gratuito
